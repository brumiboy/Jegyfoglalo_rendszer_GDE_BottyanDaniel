2026 - GDE OOP projekt feladat - Repülőjegy Foglalási Rendszer

Egyetemi beadandó projektfeladat, amely egy repülőtársaság járat- és jegyfoglalási rendszerét szimulálja objektumorientált alapokon, grafikus felhasználói felülettel.

# Funkciók
* Járatok kezelése: Belföldi és nemzetközi járatok dinamikus kezelése absztrakt alaposztály segítségével.
* Foglalások listázása: Az aktuálisan tárolt foglalások átlátható, gördíthető megjelenítése.
* Új foglalás rögzítése: Biztonságos adatbevitel dátum- és járatellenőrzéssel.
* Foglalás lemondása: Meglévő foglalások törlése utasnév és járatszám alapján.

# Technológia
* **Nyelv:** Python
* **GUI Keretrendszer:** Tkinter / TTk
